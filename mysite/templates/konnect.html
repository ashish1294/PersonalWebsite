Konnect is a mini TCP/IP stack implementation for file transfer from one system to the other over network.

This project was done as a part of the Computer Communication and Networks Course during my undergraduation in February 2014.

The objective behind this project was to learn the fundamentals of the computer networks and socket programming in C.

Implementation
Features

  • Each layer adds the corresponding header and writes the data to a temporary file which is passed to the next layer. A simple script is provided which takes the input file on the source machine and process it through each layer one my one. The same script also processes received data and processes it again through each layer to obtain original file on destination machine.

    • 3-way Handshaking in Transport layer simulated using socket programming
    • 32-bit CRC Error check in MAC Layer on sender and receiver side
    • Headers of each layer simulated using C Structures
    • In MAC layer all the data is kept in a single frame
    • Physical Layer simulated using sockets again

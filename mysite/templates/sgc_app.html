It is an android application for Amateur Astronomy Club (formerly known as Star Gazing Club), NITK Surathkal. The purpose of this application was to spread awareness about the events conducted by the club throughout the year and keep the users updated.

This application is mostly a static one with not much dynamic content as of now. It uses CSV file to store data. Different tools, features and functions provided by Android SDK were extensively used in this application.

Features

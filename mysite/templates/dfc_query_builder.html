DFC Query Builder is a mini scale database management system. Originally planned as a SQL query builder, the project evolved into something more. All major parts of a DBMS like Buffer Management, Query Optimizer, File Storage, etc. were implemented

This project was done as a part of Database Systems course during my undergraduation in November 2014

A web-based GUI has been provided using which users can build queries in a few clicks. The web server is based on PHP and interacts with DBMS server using sockets.

Features
Queries and Operators

  • Custom Block Size for each table which is always a multiple of record size which saves time and well as space in searching and storing records

    Muti-level B+ Tree for indexing based on primary key

    Temporary Tables for intermediate results

    Meta Data for each table that provides data about table

  • The Queries Supported are :
    • CREATE Table
    • SHOW Tables
    • DESCRIBE Table
    • DROP Table
    • INSERT INTO Table
    • SELECT Table
    • QUIT Server
    The Operators Supported are :
    • Greater Than or Equal (>=)
    • Less Than or Equal (<=)
    • Greater Than (>)
    • Less Than (<)
    • Equal (=)
    • Not Equal (!=)
    • Logical AND (&&)
